# ☕ Brewed & Co. // Premium Artisan Cafe Experience

A premium, highly interactive showcase website designed for an aesthetic local cafe and community workspace. **Brewed & Co.** features a beautiful menu display with dynamic cart & checkout systems, an interactive zone-based table reservation simulator with real-time seating map selection, scroll-linked animations, and an immersive background audio-only lofi experience.

This project features a high-end cream and terracotta aesthetic with frosted glassmorphic navigation headers, smooth parallax sections, cinematic coffee steam animations, and elegant typography (Cormorant Garamond & Plus Jakarta Sans).

---

# 🚀 Live Experience

An immersive, premium multi-page web application featuring:

* 💨 **Cinematic Hero Landing**: Bouncing scroll indicator, typewriter effect on headline, and custom CSS-only rising coffee steam particles.
* 📋 **Dynamic Menu Filtering**: Filter by category (Coffee, Pastries, Brunch, Tea) and dietary tags (Vegan, Dairy-Free, Gluten-Free, Nut-Free) with instant grid updates.
* 🛍️ **DTC Cart & Checkout Simulation**: Elegant drawer cart calculating item totals, free delivery progress, and simulating a secure order placement protocol with toast confirmations.
* 🪑 **Interactive Seating Map**: Zone selector (Main Hall, Window Bars, Cozy Lounge, Patio) with an interactive table selection map for reservations.
* 🎵 **Audio-only Cafe Vibes**: Floating music note player toggling a hidden YouTube embed of the client's preferred lofi track, featuring animated visualizer bars and mute/unmute control.
* 📱 **Mobile Optimized**: Sticky bottom navigation bar (Home, Menu, Reserve, About) tailored for all screen sizes.

---

# ✨ Features

* ☕ **Rising Coffee Steam**: Custom CSS animations simulating hot vapor rising above the landing hero.
* 🎯 **Add Button Feedback**: Buttons slide and show a satisfying `✓ Added!` green state on item selection.
* 📊 **Number Counter Animation**: Interactive about page section counting up statistics automatically when scrolled into view.
* 📈 **Scroll Timeline Progress**: The story timeline features a glowing vertical indicator line tracking scroll progress.
* 🖼️ **Masonry Gallery & Lightbox**: Interactive photography gallery using a masonry layout with chevron prev/next selectors and keyboard arrow support.
